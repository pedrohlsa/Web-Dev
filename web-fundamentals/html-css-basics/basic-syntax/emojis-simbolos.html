Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer forma, basta usar o par de tags <p> e </p>.

Se precisar quebrar a linha em um ponto específico
como este, utilize a tag <br>.

Exemplos de símbolos e entidades HTML: ® (Registrada) © (Copyright) ™ (Trademark) € (Euro) £ (Libra) ¥ (Iene) ¢ (Centavo) $ (Dólar) Δ (Delta) ↑ (Seta para cima)

O que é Tipografia?

A palavra vem do grego: "typos" (forma/molde) e "graphia" (escrita).

Hoje, a tipografia evoluiu e se tornou o trabalho de organizar o texto em livros ou mídias digitais.

As emoções podem ser transmitidas através do estilo das frases. Profissionalmente, você usará cores, imagens e representações gráficas para isso.

Tipos de Tipografia

Anatomia Tipográfica

Como estruturar o design das letras?

A fonte padrão do navegador geralmente varia entre Arial e Times New Roman.

O design envolve espaço vertical, linhas horizontais, maiúsculas (caixa alta), minúsculas (caixa baixa), etc.

A letra "x" é o ponto de partida para todas as outras letras minúsculas. A sua altura determina a proporção visual da fonte.

As letras possuem hastes ascendentes (que sobem, como no "b") e descendentes (que descem, como no "p"). Ambas possuem alturas específicas.

A altura da letra "M" ou "Q" geralmente define a altura das maiúsculas.

O tamanho total da linha? É chamado de "Corpo" da fonte.

Elementos anatômicos dos glifos: Divididos entre estrutura geométrica e traços humanos.

Estrutura Geométrica: A Serifa

A base ou os "pezinhos" das letras são chamados de serifas.

Por que a serifa é importante? Basicamente, ela cria uma linha visual imaginária que ajuda o cérebro a ler de forma contínua.

Nosso cérebro lê associando as palavras anteriores e posteriores. O mais importante para o reconhecimento rápido é a primeira e a última letra da palavra.

Traços Humanos da Anatomia

As letras têm componentes inspirados no corpo: haste, filete, esporão, vértice, terminal, braço, perna, coluna, barriga, olho, orelha e até cauda.

O que o filete faz? Basicamente, faz a intersecção/junção entre duas hastes ou arcos.

O conjunto de tudo isso acima é chamado de: Fonte

Uma mesma fonte pode ter variações para representar o mesmo caractere

Isso é o que chamamos de Família Tipográfica. Exemplos:

  • Light (Fina)
  • Normal / Regular
  • Semi-bold (Semi-negrito)
  • Extra-bold (Extra-negrito)

Categorias de Fontes

  • Serif (Com Serifa): Ideal para livros e textos impressos longos.
  • Sans-serif (Sem Serifa): Excelente para blocos de texto grandes em WEBSITES.
  • Monospaced (Monoespaçada): Fontes de código onde cada letra ocupa o mesmo espaço.
  • Handwriting (Cursiva/Manuscrita): Imitam a escrita à mão.
  • Display / Decorativa: Fontes comemorativas ou artísticas.

Como descrever uma fonte Display?

São fontes artísticas ou comemorativas. Basicamente, elas ignoram as regras tradicionais de anatomia e geometria em prol do estilo visual.

Como saber se uma fonte é Monoespaçada?

Todas as letras e glifos possuem exatamente a mesma largura horizontal na tela.

Para uma boa leitura, SEMPRE USE CORES COM CONTRASTE.

O padrão infalível na maioria das vezes: Texto preto no fundo branco.

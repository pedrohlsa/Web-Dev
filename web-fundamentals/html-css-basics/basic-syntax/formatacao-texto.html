Principais Formatações

Negrito e Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica), ou seja, sem significado visual apenas.

Nesta frase temos um termo de destaque usando a tag <strong> (semântica), indicando que o conteúdo é importante.

Itálico e Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica), apenas estilo visual.

Nesta frase temos um termo com ênfase usando a tag <em> (semântica), usada para dar entonação ao texto.

Texto Marcado

Podemos criar também textos marcados (tipo marca-texto) usando a tag <mark>.

Textos Grandes e Pequenos

Estamos criando um texto grande (obsoleto no HTML5) e um texto pequeno neste parágrafo.

Texto Deletado e Inserido

Podemos marcar um texto como deletado para indicar que ele foi removido ou não é mais válido.

Podemos marcar um texto como inserido para dar ênfase que ele foi adicionado posteriormente.

Também existe a tag para sublinhado, mas ela não é semântica.

Texto Sobrescrito

Para fórmulas matemáticas: $x^{20}+3$ (usando <sup>).

Texto Subscrito

Para fórmulas químicas: $H_2O$ (usando <sub>).

Endereço

A tag <address> é usada para informações de contato. Em dispositivos móveis, pode facilitar a integração com mapas:

Rua das Divisões, nº 255 - Rio de Janeiro, RJ
